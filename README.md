# Python Text-Based Adventure Game

This is a simple, text-based adventure that was built using Python. The player navigates a magical forest, makes choices, and tries to defeat a witch.

This project was created as part of the **Udacity [DECI] Level 2**.

* **Branching Story:** The player's choices directly impact the story's direction.
* **Inventory System:** The player can find and use a "magical sword" to alter the game's outcome.
* **Score System:** Actions grant or remove points. The player wins by reaching 100 points and loses if their score drops to -50.
* **Randomized Events:** Some outcomes (like running away) are randomized for replayability.
* **Play Again:** The game invites the user to play again after a win or loss.
